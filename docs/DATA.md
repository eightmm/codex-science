# Data

This repository contains no research dataset. Imported skill instructions live under `vendor/` — the K-Dense catalog as a pinned Git submodule (`vendor/scientific-agent-skills`) and the DeepMind catalog as a pinned vendored copy (`vendor/gdm-science-skills`) — plus Codex-native skills authored in `authored-skills/`. Public connector metadata lives in `connectors/public.json`.

User research data remains in the user's project. Do not copy sensitive raw data into this repository or an artifact bundle; record an approved path and hash instead.
